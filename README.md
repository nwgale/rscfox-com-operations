# rscfox-com-operations

works.rscfox.com 成果管理系统操作 Skill，支持批量上传证书 PDF + AI 自动识别回填字段。

## 支持成果类型

- 指导学生获奖
- 论文
- 著作
- 专利
- 著作权
- 个人获奖
- 纵向课题
- 横向课题

## 安装

```
请帮我 git clone https://github.com/YOUR_USERNAME/rscfox-com-operations.git ~/.workbuddy/skills/rscfox-com-operations/
```

把 `YOUR_USERNAME` 换成实际的 GitHub 用户名。

```
使用 rscfox-com-operations skill，上传 `证书目录路径` 下的 PDF，成果类别「指导学生获奖」
```

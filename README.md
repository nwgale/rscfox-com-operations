# rscfox-com-operations

works.rscfox.com 成果管理系统操作 Skill，支持批量上传证书 PDF + AI 自动识别回填字段。

## 支持成果类型

- 指导学生获奖
- 论文
- 著作
- 专利
- 著作权
- 个人获奖
- 纵向课题
- 横向课题

## 使用

复制以下文字发给 WorkBuddy：

### 第一步：安装 Skill

```
请帮我 git clone https://github.com/nwgale/rscfox-com-operations.git ~/.workbuddy/skills/rscfox-com-operations/
后续使用 rscfox-com-operations skill 进行下一步
```

### 第二步：上传

```
逐个上传 PDF 或图片，并点界面上的 AI 识别保存
文件夹地址：/Users/xxx/你的证书目录/
本次成果类别：指导学生获奖
```

（把文件夹地址和成果类别换成你自己的）
